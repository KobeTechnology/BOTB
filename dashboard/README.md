# TNS AI Dashboard (Frontend)

The TNS AI dashboard provides a user interface for monitoring how products appear across generative AI platforms.

The dashboard visualizes:

- Product visibility across AI assistants
- Brand mention rates
- Hallucination detection alerts
- AI ranking positions
- Monitoring history and metrics

## Key Features

- Monitoring overview
- Company and product selection
- Hallucination alerts
- AI response analysis
- Metrics and visibility insights

## Frontend Technology

The frontend dashboard prototype is built using **Lovable** to rapidly design a modern analytics interface.

This repository contains the backend monitoring engine that powers the dashboard.

## Backend Data Source

Dashboard data is generated from:

```
logs/monitoring_results.json
```

This file stores monitoring results generated by the TNS AI backend engine.

Each monitoring run records:

- company name
- product being evaluated
- AI assistant responses
- brand mention detection
- product ranking in AI recommendations
- hallucination detection results
- monitoring timestamp

These records allow the dashboard to visualize product visibility trends, detect misinformation, and highlight AI response accuracy across platforms such as ChatGPT, Gemini, and Claude.
